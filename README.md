# QA Dev Skills — Study Course for Nastya

A structured learning course designed for a Senior Manual QA Engineer to improve skills in general software development, backend technologies, CI/CD, and test automation.

## 🎯 Goal

Help Nastya level up from manual QA expertise into broader software development knowledge — covering HTTP, APIs, backend, CI/CD, automated testing, and developer tools.

## 📚 Course Structure

All courses are located in the `courses/` folder. Each course contains:
- **Learning Materials** — theory and explanations
- **Key Concepts** — important terms and definitions
- **Practical Exercises** — hands-on tasks to reinforce learning

## 🚀 Recommended Learning Path

1. Start with **HTTP Fundamentals** — this is the foundation for everything else
2. Move to **OpenAPI, Swagger & Postman** to understand API documentation
3. Master **Postman Collections & Environments** for practical API testing
4. Learn **Java Backend Fundamentals** to understand what happens on the server side
5. Study **Build Process** to understand how code becomes an application
6. Learn **CI/CD Fundamentals** for the big picture of modern development workflows
7. Deep dive into **Jenkins** as a practical CI/CD tool
8. Explore **Automated Testing** to understand test automation strategies
9. Study the **Deployment Process** to complete the full software lifecycle picture
10. Finally, master **Browser DevTools** for everyday QA productivity

## 💡 How to Use This Course

1. Read each course file sequentially (or follow the recommended path)
2. Try every exercise — hands-on practice is the best way to learn
3. Take notes and create your own cheat sheets
4. Ask questions when something is unclear — understanding "why" is more important than "how"
