# QA Dev Skills — Study Course for Nastya

A structured learning course designed for a Senior Manual QA Engineer to improve skills in general software development, backend technologies, CI/CD, test automation, and senior QA competencies.

## 🎯 Goal

Help Nastya level up from manual QA expertise into broader software development knowledge — covering HTTP, APIs, backend, CI/CD, automated testing, developer tools, and advanced senior QA skills including strategy, leadership, AI-powered testing, and interview preparation.

## 💡 How to Use This Course

1. Read each course file sequentially (or follow the recommended path)
2. Answer the self-check questions at the end of each course
3. Try every exercise — hands-on practice is the best way to learn
4. Take notes and create your own cheat sheets
5. Use the Russian versions (`1.dev_skills_foundation_rus/`, `2.senior_qa_competencies_rus/`, `3.deep_dive_qa_topics_rus/`) if you prefer studying in Russian
6. Ask questions when something is unclear — understanding "why" is more important than "how"
